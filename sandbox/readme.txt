sandbox
=======

Use this part of the repository to store all personal content like experiments, first prototypes, etc.

the general rule is to create a subfolder with your sourceforge user id and to place the content under this folder
e.g.

sandbox
├───mercuron
│   ├───my_hyper_cool_experiment
│   └───private_stuff
└───vineethshetty
    └───anoter_great_module
