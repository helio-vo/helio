yohkoh_sxt_trace_list YOHKOH SXT TRACE flare list

Purpose:
Common flares observed by Yohkoh/SXT and TRACE.
Description:

This list is compiled from the Yohkoh flare directories at ISAS as maintained by Mr. Sawa and the online TRACE

catalog generated by SSW. In the following table, click a line under 'SXT PFI file' (leftmost column) for a plot of

GOES/HXT light curves and SXT/TRACE data coverage.
Parameters

yst_id HEC id

Event identification number (HEC internal number)

link

Link to the SXT PFI file at ISAS.

time_start_sxt UT

Time of first flare mode SXT image.

time_end_sxt UT

Time of last flare mode SXT image.

xray_class

GOES X-ray flare class from peak GOES flux during the period the SXT flare mode images are taken.

n_img Unitless

Number of flare mode SXT images.

x_arcsec_sxt arcsec

X coordinate (in arcsec from disk center) of the center of the 6th SXT flare mode image. Picked up the 6th

because SXT sometimes changes the field of view after the first flare mode image.

y_arcsec_sxt arcsec

Y coordinate (in arcsec from disk center) of the center of the 6th SXT flare mode image. Picked up the 6th

because SXT sometimes changes the field of view after the first flare mode image.

time_start UT

First TRACE image in the period (time_start_sxt - 30min, time_end_sxt + 30min).

time_end UT

Last TRACE image in the period (time_start_sxt - 30min, time_end_sxt + 30min).

time_sxt_trace UT

Time of the TRACE image closest to time_start_sxt.

wl_dom Angstrom

Wavelength at which the greatest number of TRACE images is taken.

x_arcsec arcsec

X coordinate (in arcsec from disk center) of the center of the TRACE image at time_sxt_trace.
