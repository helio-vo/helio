stereo_euvi_catalog STEREO EUVI catalog

Purpose:
STEREO/EUVI Event Catalog (>C1 GOES class or >12 keV RHESSI)
Description:

This catalogue is a summary of the contents of the full STEREO/EUVI Event List at:



http://secchi.lmsal.com/EUVI/euvi_events.txt
Parameters

stereo_euvi_catalog HEC id

_id

Event identification number (HEC internal number)

id_num

STEREO/EUVI vent Catalogue Number.

time_start UT

Start time

time_end UT

End time

loc Unitless

Location of event in heliographic coordinates.

cadence_spacecraft s

Spacecraft cadence.

xray_class Unitless

GOES X-Ray flare class.

rhessi_peak_range keV

Peak RHESSI energy range in which event is observed.

rhessi_peak_count Unitless

Counts in peak RHESSI energy range.

rem

Comments using the following EUVI Classification:



A=Ahead only without Behind data

B=Behind only without Ahead data

I=Impulsive time profile (peak in coincidence with SXR)

P=Postflare loop or arcade (delayed emission after SXR peak)

D=Dimming (EUV flux decreases after SXR peak)

E=Eruptive filament

O=Occulted for A (if flare position is east) or B (if west)

T=Test images (or poor image quality)

W=Waves and oscillations

cme_event_source

CME reported by one or both of the following:



L=LASCO/SOHO

S=SECCHI/Cor-1,Cor-2
