wind_imf_bow_shock Wind IMF bow shock crossing

Purpose:
List of WIND IMF Bow Shock crossing
Description:

The event list provides a list of bow shock (BS) crossing times. For each crossing, the following times are given:



The time of the last BS crossing as the spacecraft was leaving the Earth magnetosheath.

The first full hour which has only unshocked solar wind data in it.

The last full hour which still contains interplanetary magnetic field data.

The time of the 1st BS crossing on the way into the Earth magnetosphere.
Parameters

wind_imf_bow_sho HEC id

ck_id

Event identification number (HEC internal number)

last_bs_crossing UT

The time of the last BS crossing as the spacecraft was leaving the Earth's magnetosheath.

last_bs_flag

A flag to indicate one of the following for the last BS crossing time:



#: First data point in our data set

*: BS occurred in data gap, time listed is the last point before the gap

+: Strong foreshock activity before BS also excluded

first_hr_kept UT

The first full hour which has only unshocked solar wind data in it.

first_hr_flag

A flag to indicate one of the following for the last BS crossing time:



+: Strong foreshock activity before BS also excluded

last_hr_kept UT

The last full hour which still contains interplanetary magnetic field data.

last_hr_flag

A flag to indicate one of the following for the last BS crossing time:



+: Strong foreshock activity before BS also excluded

first_bs_crossing UT

The time of the 1st BS crossing on the way into the Earth magnetosphere.

first_bs_flag

A flag to indicate one of the following for the last BS crossing time:



#: First data point in our data set

*: BS occurred in data gap, time listed is the last point before the gap

+: Strong foreshock activity before BS also excluded

time_start UT

Start time set as the last BS crossing time.

time_end UT

End time set as the time of the 1st BS crossing on the way into the Earth magnetosphere.
